Sistema de Posto de Combustível
Sistema desktop de gerenciamento de posto de combustível, desenvolvido em Python com Tkinter e PostgreSQL.
Descrição Geral
O sistema permite gerenciar os combustíveis disponíveis no posto, as bombas de abastecimento e os registros de abastecimentos realizados. Conta com duas visões: a do Atendente (operacional) e a do Administrador (gerencial completo).
Funcionalidades

Cadastro e gerenciamento de combustíveis (tipos, preço por litro e estoque)
Cadastro e gerenciamento de bombas (número, combustível associado e status)
Registro de abastecimentos com suporte a três modalidades de preço
Listagem de abastecimentos com filtro por combustível e data
Controle automático de estoque ao realizar abastecimentos
Tela Sobre com informações do sistema e do autor

Tecnologias

Python 3.13
Tkinter (interface gráfica)
PostgreSQL 18 (banco de dados)
psycopg 3 (driver de conexão)
